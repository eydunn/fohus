# fohus

Webscraper and data collector of realestate data in the Faroe Islands
https://fohus.now.sh/

> GET /api

Returns an array of objects from

* Betriheim.fo
* Skyn.fo
* Meklarin.fo

#### Properties:

* status

  * new
  * bid
  * openhouse
  * sold
  * none

* **address** (string)
* **area** (string)
* **img** (string / url)
* **price** (string)
* **url** (string/url)
* **provider** (string)
